Draw Me
=======

A silly little program that will render an image in a terminal with two horizontal pixels per character.

<img src="http://i.imgur.com/6ulC2sY.png">

How to use:

* `make`
* `./draw_me <image>`

OSX homebrew dependencies:

* pkg-config
* imagemagick

Debian dependencies:

* libmagickwand-dev
